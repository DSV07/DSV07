## 🧩 Projetos em destaque

### 💰 Plutus — Plataforma de Gestão Financeira

Sistema web voltado para **organização financeira pessoal e controle financeiro para pequenos negócios**, com foco em entradas, saídas, categorias, acompanhamento de gastos e visualização de dados financeiros.

**Tecnologias:** React, Node.js, JavaScript, MySQL, APIs.

---

### 📅 Vantage — Sistema de Agendamento

Solução desenvolvida para **gestão de horários, serviços e clientes**, pensada principalmente para negócios locais como barbearias, salões e prestadores de serviço que precisam organizar sua agenda de forma simples e eficiente.

**Tecnologias:** React, Node.js, JavaScript, MySQL.

---

### 💇‍♀️ Flowbraids — Agendamento para Trancistas

Sistema voltado para **profissionais de tranças**, permitindo organizar atendimentos, serviços, horários disponíveis e informações de clientes, com foco em praticidade e gestão profissional da agenda.

**Tecnologias:** React, Node.js, JavaScript, MySQL.

---

### 📦 Stockin — Sistema de Controle de Estoque / ERP

Sistema web desenvolvido para **controle de produtos, organização de registros e apoio à gestão operacional básica**, evoluindo de um controle de estoque para uma proposta de mini ERP para pequenos negócios.

**Tecnologias:** React, Node.js, JavaScript, MySQL.

---

### 📝 Summarizin — Sistema de Resumos e Organização de Conteúdo

Aplicação voltada para **resumir, organizar e estruturar conteúdos extensos**, como aulas, textos, documentos e materiais de estudo, facilitando a revisão e o acesso às informações principais.

**Tecnologias:** React, Node.js, JavaScript, APIs, Inteligência Artificial.

---

### 🌐 Landing Pages e Sites Institucionais

Desenvolvimento de páginas responsivas com foco em apresentação profissional, usabilidade e captação de contatos.

**Tecnologias:** HTML, CSS, JavaScript e React.

---

### 🗄️ Sistemas Web com Banco de Dados
